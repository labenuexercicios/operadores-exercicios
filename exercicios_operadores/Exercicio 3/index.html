Escreva as operações lógicas abaixo em formato de código. Imprima no console se o valor da operação é verdadeira ou falsa:

  • a) 5 é maior que 20 e também é menor que 2;
  • b) 5 é igual a 5 ou é igual a “5”;
  • c) negação de (vinte é maior que cinquenta)
  • d) negação de (vinte é maior que cinquenta OU cinquenta é maior que sessenta)
Exemplo: const operacao1 = 20 < 50 && 50 > 90; console.log(operacao1); /*false, porque 50 não é maior que 90, consequentemente temos true && false, que devolve false*/
